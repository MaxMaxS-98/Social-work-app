# NoSQL Challenge: Social Network API
 
## Description

This is a social network API built with Express.js for a social network web application where users can share their thoughts, react to friendsâ€™ thoughts, and create a friend list. This application uses a NoSQL database so the website can handle large amounts of unstructured data.

 User can update a user.
 User can update a thought.
 User can delete a user.
